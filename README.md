# DevConnector MERN Project

### Social Network Web Application for Developers
Small MERN stack social media application which includes authentication, profiles, and forum posts.
# Technology
* MongoDB
* Express.js
* React.js
* Node.js
* Javascript
* Redux
* Postman
* Heroku (For deployment)
